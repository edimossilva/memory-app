# install

1. build containers

   `docker-compose build`

1. start containers in background

   `docker-compose up -d`

1. config db

   `docker-compose exec memory_app_back rake db:create`

   `docker-compose exec memory_app_back rake db:migrate`

1. tests

   `docker-compose exec memory_app_back rspec`

1. access

   `http://localhost:3333`
